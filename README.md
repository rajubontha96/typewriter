# 🚀 Touch Typing Trainer (Full Stack)

A full-stack web application to practice touch typing using home row keys (`asdfjkl;`).  
Built with **React, Redux, Redux-Saga, Node.js, Express, and SQLite**.

## 🧠 Features

### ✅ Core Features
- ⌨️ Typing practice (home row keys)
- 🎯 Real-time key highlighting
- ⏱️ 5-minute timer
- 📊 Accuracy & WPM calculation

### 🚀 Advanced Features
- 🔐 JWT Authentication (Login & Signup)
- 🧠 Redux + Redux-Saga state management
- 🎮 Streak & Level system (Beginner → Pro)
- 📈 Performance tracking
- 🏆 Leaderboard (Top scores)

### 🎨 UI/UX
- 🌙 Modern dark theme
- 💎 Glassmorphism design
- 🎹 Keyboard visualization
- ⚡ Real-time feedback (Correct / Wrong)
